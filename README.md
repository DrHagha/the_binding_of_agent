# The Binding of Agent

## Setup

- **Unity Version**: `6000.3.8f1`
- Unity Hub에서 이 폴더를 프로젝트로 열어주세요.
- 최초 실행 시 패키지 다운로드에 시간이 걸릴 수 있습니다.

## Project Structure

```
Assets/
  Scripts/    - C# 스크립트 (전투 시스템, UI 등)
  Scenes/     - BattleScene
  Resources/  - 토큰 이미지, 폰트, TMP 설정
  Shader/     - TextMeshPro 셰이더
  Sprite/     - 스프라이트 에셋
```
